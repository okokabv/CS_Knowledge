# 1. 계정계(Core Banking):          
   - 고객의 거래 데이터를 처리하는 핵심 시스템         
   - 계정계 시스템: 공통, 수신, 여신, 신탁, 보험, 카드, 외환, 대행업무 등으로 구성          
   계정(통장, 계좌 등)을 관리하는 시스템들을 '계정계'라고 함           
   금액 정보 등 매우 중요한 시스템이기 때문에, 보수적으로 운영       
   많은 데이터량 (계정당 1억 건은 가뿐히 넘음)      

# 2. 정보계:
     - 각 금융기관의 대·내외망을 연결하는 시스템
